Website - Sales Order Delivery Date

Key Features:

  • On eCommerce allows you to see that in how many days the product will be delivered based on days configure
  • On Cart user can see the delivery date of product
  • Generate delivery order based on the product delivery date

Overview

  • From Product menu, under Inventory Tab user can configure Customer Lead Time.
  • On eCommerce Shop page user can see in how many days the product will deliver based on customer lead time configure. Also option for user to hide the delivery days information.
  • In Product detail page also information for product deliver days.
  • Into cart, it shows when the product will be delivered.
  • In Sale order line user can see the delivery date of product.
  • Sales Delivery Order is generate based on the delivery date of product.
